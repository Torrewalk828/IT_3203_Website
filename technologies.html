HTML

HTML (Hypertext Markup Language) structures web pages using elements such as headings, paragraphs, links, and images.

CSS

CSS (Cascading Style Sheets) controls the appearance of web pages including layout, colors, fonts, and spacing.

HTTP

HTTP allows browsers and servers to communicate. Browsers send requests and servers respond with web pages.
